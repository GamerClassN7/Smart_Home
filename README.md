# API
### POST Message (Spínač)
```
{
	"token":"2"
}
```

### Answer (Spínač)
```
{
	"device":{
		"hostname":"2",
		"sleepTime":0
		},
		"state":"succes",
		"value":"0"
	}
}
```

### POST Message (Sensor)
```
{
	"token":"4",
	"values":{
		"door":{
			"value":1
		}
	}
}
```
Answer (Sensor)
```
{

}
```
